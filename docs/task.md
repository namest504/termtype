# CLI 타이핑 연습 프로그램 개발

## 1단계 ~ 11단계 (생략)

## 12단계: 테마 디자인 개선 및 버그 수정

- **Hex 테마 개선**
  - 화면 전체에 임의의 헥스 코드를 채우고, 중앙에서 타이핑을 시작하도록 변경하여 몰입감 증대
- **커서 위치 버그 수정**
  - `code` 테마: 특수문자를 포함한 문자열의 인덱싱 문제로 발생하던 커서 밀림 현상 수정
  - `diff` 테마: 미세한 계산 오류로 커서가 한 칸씩 밀리던 문제 수정
- **빌드 오류 수정**
  - `code` 테마 수정 중 발생한 문법 오류 및 `diff` 테마 커서 수정 누락 문제 해결
- **커서 위치 버그 재수정**
  - `code` 및 `diff` 테마의 커서 위치 수정이 잘못되었던 문제를 사용자 피드백을 바탕으로 다시 수정
